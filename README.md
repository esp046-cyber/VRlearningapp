# VRlearningapp
VR Learning System mobile app with:
Features:

Small title font (1.2em) as requested
1% margins throughout the layout
VR-themed design with purple gradient background and glassmorphic cards
No 3D elements or search bar as specified
Course catalog with progress tracking
Lesson breakdown for each course
Stats dashboard showing courses, completion, and time spent
Tab navigation (Explore, In Progress, Completed)
Floating action button to launch VR mode
Mobile-optimized with responsive grid layout

The app uses a modern glassmorphism aesthetic with vibrant gradients, perfect for a futuristic VR learning platform. Each course can be tapped to view its lessons, and lessons can be launched in VR mode. The interface is clean, immersive, and designed specifically for mobile viewing.
